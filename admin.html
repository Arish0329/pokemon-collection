102 張卡牌管理

GitHub Pages 部署教學

1 安裝 GitHub Desktop
前往 desktop.github.com 下載並安裝 GitHub Desktop，然後登入你的 GitHub 帳號。
2 建立新 Repository
點選 File → New Repository，命名為 pokemon-collection，本地路徑選擇這個資料夾（pokemon-collection/）。
3 Commit 初始檔案
左側會列出所有新增的檔案。在 Summary 欄位輸入 初始提交，按下 Commit to main。
4 Publish Repository
點選右上角 Publish repository，確認 Public（公開），按下 Publish。
5 開啟 GitHub Pages
到 GitHub 網站 → 你的 repository → Settings → Pages → Source 選 Deploy from a branch → main → / (root) → 儲存。
6 等待部署完成
約 1–2 分鐘後，網址會是：
https://你的帳號.github.io/pokemon-collection/
7 以後新增掃描圖片的流程
① 在本後台上傳圖片並套用 → ② 開啟 GitHub Desktop → ③ 圖片和 data.json 會出現在變更清單 → ④ Commit → ⑤ Push → ⑥ 網站自動更新。

移除收藏確認

確定要將此卡牌從收藏中移除嗎？圖片檔案需要手動從 images/ 資料夾刪除。

儲存 data.json

點擊「下載 data.json」，將檔案覆蓋到專案資料夾中，然後透過 GitHub Desktop commit 並 push。
